# 💖 For You, Always

A soft, emotional, and interactive web experience designed to express feelings that are hard to say out loud.

This project is more than just a webpage — it’s a **digital letter** enhanced with animations, interactivity, and immersive design to create a meaningful moment.

---

## ✨ Features

* 💌 Real-time typing animation for emotional storytelling
* 💫 Floating emojis and sparkle effects
* 💓 Heartbeat animation synced with key emotional words
* 🎨 Modern UI with gradients, glassmorphism, and smooth transitions
* 💖 Interactive choices (forgiveness flow)
* 🌧️ Dynamic visual effects (hearts, sparkles, broken hearts)
* 📱 Fully responsive design for all devices

---

## 🧠 Concept

This project represents a **memory and apology experience**:

* Starts with a calm landing screen
* Transitions into a “memory mode”
* Displays a heartfelt typed message
* Ends with a choice-based emotional interaction

It is designed to feel **personal, slow, and emotionally engaging**.

## 🎮 How It Works

1. Click **"Hey… please click this 💌"**
2. Enter the memory experience
3. Watch the message get typed in real-time
4. Choose your response:

   * 💖 Forgive
   * 💔 Still mad

Each choice triggers a different emotional animation.
